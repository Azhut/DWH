# 🚀 Профилирование производительности

## 📋 Как использовать

### 1. Включите профилирование
В `.env` файле добавьте:
```
ENABLE_PROFILING=true
```

### 2. Запустите приложение
```bash
python main.py
```

### 3. Загрузите файлы через API
Используйте POST `/api/v2/upload`

### 4. Посмотрите результаты
```bash
python scripts/simple_profiling_report.py
```

## 🎯 На что обратить внимание

- **ProcessSheetsStep > 2с** - главный кандидат на оптимизацию
- **ExtractDataStep > 2с** - вторая цель
- **Память > 100MB** - проверьте утечки

## 📁 Структура файлов

```
profiling_data/           # Данные профилирования (автоматически)
scripts/
└── simple_profiling_report.py  # Отчеты с текстовой визуализацией
```

## 🎨 Визуализация

Отчет включает:
- **Текстовые гистограммы** времени выполнения
- **Гистограммы потребления памяти**
- **Временную шкалу** выполнения шагов
- **Детальную статистику** и рекомендации

## 🛑 Отключение

Удалите `ENABLE_PROFILING=true` из `.env` или замените на `false`.
